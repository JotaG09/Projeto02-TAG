## Como Executar
### 1. Clonar o repositório

```bash
git clone https://github.com/JotaG09/Projeto02-TAG.git
cd Projeto02-TAG
```

### 2. Compilar o Projeto
```bash
make
```

### 3. Compilar e Executar Direto

```bash
make run
```

### 4. Limpar arquivos temporários

```bash
make clean
